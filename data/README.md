# Data Directory Structure

This directory contains generated examples for both inference and training visualizations.

## Directory Structure

```
data/
├── inference/           # Inference examples
│   ├── examples.json   # Index file for inference examples
│   ├── en/             # English examples
│   ├── cs/             # Czech examples
│   ├── fr/             # French examples
│   └── ...             # Other languages
│
└── training/           # Training examples
    ├── examples.json   # Index file for training examples
    ├── en/             # English examples (if applicable)
    └── ...             # Other languages
```

## File Types

### Inference Examples
- Generated by `generate_data.sh`
- Uses `llm_inference_server.py` and `llm_inference_client.py`
- Files are organized by language and variant (greedy, sampling, random)
- Example: `data/inference/en/en-001-greedy.json`

### Training Examples
- Generated by `generate_training_data.sh`
- Uses `llm_training_server.py` and `llm_training_client.py`
- Files are organized by language and type (full, limited)
- Example: `data/training/en/en-001-full.json`

## Index Files

Each subdirectory (`inference/` and `training/`) has its own `examples.json` file that serves as an index:
- Lists all available examples in that subdirectory
- Contains metadata (language, model, parameters, etc.)
- Automatically generated by `create_examples_index.py`

## Generating Examples

### Inference
```bash
bash generate_data.sh
```

### Training
```bash
bash generate_training_data.sh
```
